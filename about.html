GitHub搭建自己的博客

    概述

    一直想弄一个自己的博客，抱着这个想法就开始在网上找创建博客的方法，本人是个学生党，没多少钱，伴着这个想法就在网上找了大佬的教程，自己在GitHub上建了个免费的博客来玩玩，步骤也是很简单的！

    准备工作

    还是学着大佬们把GitHub是什么讲一讲,我们主要是要利用GitHub中的GitHub pages.

    GitHub是什么：

    GitHub是一个利用Git进行版本控制、专门用于存放软件代码与内容的共享虚拟主机服务,很多人都把它称作程序员的同性交友网站,具体为啥这么叫我也不知道，这个文字也是用大佬的话写的！

    GitHub pages是什么：

    Github Pages设计的初衷是为托管在GitHub上的项目提供介绍页面,开发者们可以通过GitHub Pages为他们的每一个项目创建一个用于介绍该项目的静态网站,不过由于他的空间免费而且稳定,因此用它搭建一个个人博客网站是再好不过了.！，不过我上网查了下它的大小只有1个G，所以做个自己玩的博客还是绰绰有余了，如果你想做更大的网站，就必须得去买虚拟机了，和域名了。

    Git是什么：

    Git是一个开源的分布式版本控制系统,可以有效、高速的处理从很小到非常大的项目版本管理.它的作用 和Svn类似,就是一个版本控制的工具,用它可以将我们写的代码提交到Github上.
